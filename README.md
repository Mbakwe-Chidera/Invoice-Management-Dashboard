# Invoice Management Dashboard – Pure HTML & CSS

A full-page invoice list with sidebar nav, live stats, search and filter bar.  

---

## What you see
- **Collapsible sidebar** – logo + 5 nav links (Dashboard, Invoices, Clients, Reports, Settings)  
- **Top filter bar** – status picker, date range, min/max amount, Apply button  
- **KPI cards** – Total Revenue, Pending, Overdue  
- **Search bar** – filters the list below (placeholder)  
- **Invoice table** – 25 sample rows: ID, client, amount, date, colour-coded status badge (Paid, Pending, Overdue)
